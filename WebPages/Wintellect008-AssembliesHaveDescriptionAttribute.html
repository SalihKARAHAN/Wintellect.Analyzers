WINTELLECT008: Assemblies have DescriptionAttribute Analyzer

TypeName

 AssembliesHaveDescriptionAttributeAnalyzer

CheckId

WINTELLECT008

Category

Design

Breaking Change

Non Breaking
Cause

The assembly being analyzed does not have a DescriptionAttribute set at the assembly level. Either it is missing or the string is empty.

Rule Description

The description on an assembly can help identify an assembly's purpose. All assemblies you build need to have the DescriptionAttribute set at the assembly level.

How to Fix Violations

Somewhere in the assembly code, generally in a AssemblyInfo.cs file, ensure you have the AssemblyDescriptionAttribute set to a valid string.

When to Suppress Warnings

Do not suppress a warning from this rule.

Example

The following example how to set the AssemblyDescriptionAttribute.

C# 
using System.Reflection;

[assembly: AssemblyDescription("Roslyn Analysis and Code Fixes to assist with better development")]
